# LensLink

이미지를 업로드하면 OpenAI가 의류 정보를 분석하고,
여러 쇼핑몰에서 동일하거나 유사한 상품을 검색하는
이미지 기반 상품 검색 백엔드 프로젝트입니다.

# 기술 스택
- Java 21
- Spring Boot
- Spring WebFlux(WebClient)
- OpenAI API
- Jackson
